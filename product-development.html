Product Development – Samples

Currently, I lead the product development for PitchBook -- my current company’s fastest growing product. This page shows four major features delivered in 2019.

General Layout

The web platform was launched in early 2019 and shows the equity data of companies. Working with the developers and UX designers, I wrote the technical specifications on the API and data feed that powers the platform and the layout/categories. Parameters such as dates, report types, period ends, currency, units, and other metadata were carefully determined to ensure the datapoints are selected appropriately. I also designed the presentation layout so users can quickly navigate to the important items.

Drill Down

Drill Down is the core value-added feature that differentiates PitchBook from its competitors. Clicking on a number will show the calculation formula and inputs, which is highly invaluable for clients as it provides data transparency. Inputs can also be further "drilled" to show the child-inputs used to determine the parent value.

Source Document Tagging

Clicking the little document icon to the right of the input in the previous image (997,000) allows us to open the source document and highlight the value that was used. This allows the client to perform their due diligence efficiently to ensure our data is correct and which financial report was used. This was a huge breakthrough as I encountered various OCR issues in previous projects.

LaTeX for Complicated Formulas

The next part was a creative solution to handle complicated formulas. The calculation in the image is Degree of Operating Leverage, which has a formula that has a fraction within a fraction. The left shows how it would have looked if this feature is absent. Using a LaTeX library to render the formula, the display becomes much more elegant.
